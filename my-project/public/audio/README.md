# Audio Files Directory

Add your audio files (MP3, WAV, OGG) to this directory to use them in your music app.

## Supported Formats:
- MP3 (most compatible)
- WAV (high quality, larger files)
- OGG (good compression)
- M4A (Apple format)

## Example Files:
- midnight-city.mp3
- neon-lights.mp3
- digital-rain.mp3
- coffee-shop-dreams.mp3
- mountain-road.mp3
- starlight-serenade.mp3

## How to Add:
1. Copy your audio files to this directory
2. Update the `audio_url` field in `db.json` to `/audio/filename.mp3`
3. Restart your development server if needed

## File Naming:
- Use lowercase names
- Replace spaces with hyphens
- Avoid special characters
